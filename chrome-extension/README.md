# A股每日复盘提醒 Chrome 插件

这是一套基于 Chrome Manifest V3 的原生浏览器插件，当前主定位是“收盘后复盘打分助手”。它保留了轻量化行情查看和自选股管理，但核心目标是稳定完成每日提醒、规则评分、报告生成与导出。整个工程不依赖任何第三方前端框架，数据全部通过 `chrome.storage.local` 本地持久化。

## 二、核心功能说明

### 1. 每日复盘定时提醒

- 默认提醒时间为 `15:30`
- 使用 `chrome.alarms` 注册每日复盘提醒
- 使用 `chrome.notifications` 发送桌面通知
- 当开启“仅交易日提醒”时：
  - 周末自动跳过
  - 工作日尝试通过上证指数最新日线判断是否休市
  - 支持在设置页手工维护节假日例外

### 2. 自定义条件预警

- 预设规则：
  - 跌破 5 日均线
  - 跌破 BBI
  - 单日涨跌幅超阈值
  - 放量/缩量
  - 触及止盈止损
- 规则支持启用、停用、删除
- 后台轮询更新数据后自动校验
- 触发后推送桌面通知，并在控制台记录预警日志

### 3. 自定义规则打分

- 预设 5 条打分规则，每条默认 1 分
- 支持自定义分值
- 每次执行复盘时自动给全部自选股评分
- 生成排序报告，展示：
  - 总分
  - 命中的加分项
  - 未命中的扣分项
  - 现价、涨跌幅、换手额、成交量等

### 4. 行情数据

- 实时行情：新浪财经行情接口
- 历史日线：东方财富日 K 接口
- 缓存策略：
  - 实时行情采用批量请求 + 短时缓存
  - 历史日线采用分股票长缓存
  - 请求失败时优先回退旧缓存，避免页面空白
  - 同时只允许一个刷新任务运行，避免重复访问免费接口
- 推荐场景：
  - 收盘后复盘
  - 次日开盘前查看前一交易日打分
  - 规则对比和报告导出
- 自动计算：
  - MA5 / MA10 / MA20
  - BBI
  - MACD
  - 成交量比较

## 三、安装教程

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录下的 `chrome-extension` 文件夹
5. 加载成功后，将插件固定到工具栏

## 四、基础使用教程

### 1. 快速盯盘

1. 点击浏览器右上角插件图标
2. 在弹窗中查看自选股、涨跌情况和今日复盘排行
3. 点击“打开完整面板”进入大屏盯盘界面

### 2. 添加自选股和持仓

1. 打开完整控制台页面
2. 在左侧输入：
   - 股票代码
   - 股票名称
   - 成本价
   - 持仓数量
   - 止盈价
   - 止损价
3. 点击“添加/覆盖自选股”

### 3. 查看 K 线

1. 在左侧自选股列表中点击“查看”
2. 主面板会展示对应股票的 K 线、MA5、BBI 以及当前关键数据

### 4. 手动复盘

1. 在弹窗或控制台点击“复盘”
2. 插件会自动生成当日评分报告
3. 排行结果会展示在弹窗和控制台中
4. 在控制台中可一键导出 `Markdown` 或 `HTML` 报告

## 五、自定义规则配置教程

### 1. 配置提醒规则

1. 进入设置页
2. 在“自定义条件预警”模块填写：
   - 规则名称
   - 规则类型
   - 周期
   - 阈值
   - 是否启用
3. 点击“新增预警规则”

### 2. 配置打分规则

1. 进入设置页
2. 在“自定义打分规则”模块填写：
   - 规则名称
   - 规则类型
   - 分值
   - 周期
   - 阈值
   - 是否启用
3. 点击“新增打分规则”

### 3. 常见规则示例

- `价格跌破均线` + 周期 `5`
- `单日绝对涨跌幅大于等于阈值` + 阈值 `5`
- `成交量较前日放量倍数` + 阈值 `1.5`
- `MACD 金叉`
- `收盘价大于开盘价`

## 六、关键代码说明

### 1. `background/service-worker.js`

负责后台调度，是整个插件的核心中枢：

- 初始化默认数据
- 创建轮询与复盘提醒闹钟
- 拉取行情
- 评估预警
- 生成复盘报告
- 推送桌面通知

### 2. `shared/market-api.js`

封装行情获取逻辑：

- `fetchSinaQuotes` 获取实时行情
- `fetchEastMoneyHistory` 获取历史日 K
- `fetchMarketBundle` 批量获取行情快照
- `deriveStockSnapshot` 生成规则引擎统一消费的数据结构

### 3. `shared/indicators.js`

封装技术指标：

- `calcMA`
- `calcBBI`
- `calcMACD`
- `attachIndicators`

后续新增 RSI、KDJ、布林带时，建议直接在这里扩展。

### 4. `shared/rules.js`

封装规则引擎：

- `evaluateRule` 单规则判断
- `evaluateAlerts` 生成预警命中列表
- `evaluateScores` 生成打分结果

后续新增规则时，只需要：

1. 在 `shared/defaults.js` 的 `RULE_TYPES` 增加新规则定义
2. 在 `shared/rules.js` 的 `evaluateRule()` 里新增对应分支
3. 如果规则依赖新指标，再去 `shared/indicators.js` 扩展指标计算

## 八、注意事项

- 新浪和东方财富接口属于公开可访问的免费接口，稳定性较好，但不排除偶发限流或格式调整
- 当前版本为了轻量化，轮询频率默认 5 分钟；若要更高频，请自行在设置中调整
- 若准备上架 Chrome 应用商店，建议补充隐私声明、截图、图标与更完整的异常兜底策略
