# A股每日复盘提醒 Chrome 插件

基于 Chrome Manifest V3 的原生浏览器插件，定位为"收盘后复盘打分助手"。支持自选股管理、每日定时提醒、规则评分与报告生成。不依赖任何第三方前端框架，数据全部通过 `chrome.storage.local` 本地持久化。

## 目录结构

```text
├─ manifest.json
├─ assets/icons/
├─ background/service-worker.js
├─ dashboard/
├─ options/
├─ popup/
└─ shared/
   ├─ chart.js
   ├─ defaults.js
   ├─ indicators.js
   ├─ market-api.js
   ├─ report.js
   ├─ rules.js
   ├─ storage.js
   ├─ ui.js
   └─ utils.js
```

## 安装

1. 打开 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本仓库根目录
4. 将插件固定到工具栏

## 核心功能

### 自选股管理

- 支持 A 股、港股、北交所
- 弹窗搜索添加（代码 / 名称 / 拼音简写）
- 按得分、涨跌幅排序
- 点击卡片展开 K 线详情面板，再次点击或点空白关闭

### K 线详情面板

- 覆盖式弹出，不撑宽弹窗
- 展示现价、涨跌幅、MA5、BBI、换手率
- 45 日 K 线图（含 MA5 / BBI 均线、成交量柱）
- 当日加分项 / 减分项规则命中详情

### 每日复盘打分

- 预设打分规则，每条默认 1 分，支持自定义分值
- 规则类型包括：
  - 收盘价高于昨日收盘
  - 不破 BBI
  - 无巨量阴线
  - **趋势向上（线性回归）**：取最近 N 日收盘价做最小二乘回归，斜率达标且 R² ≥ 阈值才判为趋势向上，有效过滤横盘震荡股
  - KDJ 未死叉
  - MACD 金叉 / 死叉
  - 均线多头排列等

### 定时提醒

- 默认 15:10 / 16:15 自动刷新行情并生成复盘报告
- 使用 `chrome.alarms` + `chrome.notifications` 推送桌面通知
- 支持手工维护节假日例外

### 行情数据

| 数据 | 来源 |
|------|------|
| 股票搜索 | 新浪财经 suggest |
| 实时行情 | 东方财富个股接口 |
| 历史日 K | 东方财富 push2his |
| 分时数据 | 东方财富 push2 |

缓存策略：历史日线 6 小时 TTL，请求间隔 220ms 防限流，失败时回退旧缓存。

## 规则引擎扩展

新增规则只需三步：

1. 在 `shared/defaults.js` 的 `RULE_TYPES` 加新类型
2. 在 `shared/rules.js` 的 `evaluateRule()` 加对应 case
3. 若依赖新指标，在 `shared/indicators.js` 扩展后挂到 `attachIndicators()`
