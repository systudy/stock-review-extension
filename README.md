# A股每日复盘提醒 Chrome 插件

基于 Chrome Manifest V3 的原生浏览器插件，定位为"收盘后复盘打分助手"。支持自选股管理、K 线查看、规则评分、每日提醒与报告导出。不依赖任何第三方前端框架，数据全部通过 `chrome.storage.local` 本地持久化。

## 目录结构

```text
chrome-extension/
├─ manifest.json
├─ assets/icons/
├─ background/
│  └─ service-worker.js      # 后台调度中枢
├─ dashboard/                # 完整盯盘面板
├─ options/                  # 规则设置页
├─ popup/                    # 工具栏弹窗
└─ shared/
   ├─ chart.js               # Canvas K线渲染
   ├─ defaults.js            # 规则类型枚举与默认配置
   ├─ indicators.js          # MA / BBI / MACD 计算
   ├─ market-api.js          # 新浪/东财行情接口
   ├─ report.js              # Markdown / HTML 报告生成
   ├─ rules.js               # 规则引擎
   ├─ storage.js             # storage 读写封装
   ├─ ui.js                  # 主题等 DOM 辅助
   └─ utils.js               # 日期、数字、交易日工具
```

## 安装

1. 打开 `chrome://extensions/`，开启右上角**开发者模式**
2. 点击**加载已解压的扩展程序**，选择 `chrome-extension/` 文件夹
3. 将插件图标固定到工具栏

## 主要功能

### Popup 弹窗
- 自选股列表，2 列紧凑卡片，显示股价、涨跌幅、复盘得分
- 点击任意股票，右侧展开 K 线图、5 项关键指标与规则详情
- 顶部操作栏：↻ 刷新复盘 / ⚙ 规则设置 / ▣ 完整面板 / ＋ 添加股票

### 每日复盘定时提醒
- 默认 15:10 / 16:15 两个时间点自动触发
- 使用 `chrome.alarms` 调度，`chrome.notifications` 推送桌面通知
- 支持"仅交易日提醒"，可手工维护节假日例外

### 自选股管理
- 支持按代码、名称或拼音简写搜索添加
- 支持批量导入（每行一只，格式：代码或"代码 名称"）
- 删除时有确认弹窗防误操作

### 规则打分
- 内置多条打分规则，每条支持自定义分值和启用/停用
- 复盘时自动对全部自选股评分，生成得分排行
- 弹窗卡片直接显示得分，高分红色、低分灰色
- 规则命中/未命中详情可在 K 线面板内折叠查看

### 条件预警
- 支持：跌破均线、跌破 BBI、涨跌幅超阈值、放量/缩量、触及止盈止损
- 后台轮询触发后推送桌面通知并记录预警日志

### K 线与指标
- Canvas 渲染日 K 线，叠加 MA5 / MA10 / BBI
- 行情来源：新浪财经（实时）+ 东方财富（历史日 K）
- 请求失败自动回退缓存，避免页面空白

### 报告导出
- 完整面板支持一键导出当日复盘报告（Markdown / HTML）

## 使用说明

### 添加自选股
点击弹窗顶部 **＋** 按钮，输入代码、名称或拼音简写，从搜索结果中选择后确认添加。

### 批量导入
完整面板 → **批量导入自选股**，每行粘贴一只股票（支持格式：`600519`、`600519 贵州茅台`、`00700.HK`）。

### 查看 K 线
点击弹窗中的任意股票卡片，右侧展开该股票的 K 线图和指标详情；再次点击同一卡片收起。

### 手动复盘
点击弹窗顶部 **↻** 按钮，触发行情刷新和规则重新评分，完成后自动更新所有卡片得分。

### 自定义规则
点击 **⚙** 进入规则设置页，可新增、编辑、启用/停用预警规则和打分规则。

## 扩展说明

新增规则类型：
1. 在 `shared/defaults.js` 的 `RULE_TYPES` 添加定义
2. 在 `shared/rules.js` 的 `evaluateRule()` 添加对应分支
3. 如依赖新指标，在 `shared/indicators.js` 扩展后调用 `attachIndicators()`

## 注意事项

- 新浪和东方财富接口为公开免费接口，偶发限流时自动回退缓存
- 默认轮询间隔 5 分钟，可在设置中调整
- 如需上架 Chrome 应用商店，请补充隐私声明和完整异常兜底
